### 기능 구현 목록<br>

1. 영화 선택 정보 출력<br>
2. 예약할 영화 선택<br>
3. 인원선택<br>
4. 예매 가능 유무 확인<br>
   1) 상영시작 시간이 지난 경우<br>
   2) 예매 간으 인원을 초과하는 경우<br>
   3) 두 영화의 시간차이가 1시간 이내가 아닌 경우<br>
   4) 상영목록에 없는 영화를 선택한 경우<br>
5. 추가 예약 시 2 ~ 3 반복<br>
6. 결재 시<br>
   1) 신용카드 5%할인<br>
   2) 현금결제 2%할인<br>
   3) 할인은 포인트를 제외한 금액에서 적용<br>
7. 최종 예매금액<br>
